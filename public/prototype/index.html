📋 Cara Menggunakan:

  • Klik link di bawah untuk membuka setiap screen
  • Copy seluruh HTML (Ctrl+U lalu Ctrl+A, Ctrl+C)
  • Buka Figma → Install plugin "html.to.design"
  • Paste HTML → Convert ke Figma layers
  • Edit dan customize di Figma sesuai kebutuhan

🎨 Design System:

  • Primary Color: #0A9294 (Hijau Sage)
  • Secondary Color: #067678 (Darker Sage)
  • Font: Open Sans, Segoe UI
  • Border Radius: 6px (buttons), 10px (cards)
  • Shadows: 0 2px 10px rgba(0,0,0,0.08)
  • Badge Colors: Success (#d4edda), Warning (#fff3cd), Danger (#f8d7da)
